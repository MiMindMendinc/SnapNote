# SnapNote

Offline photo-to-notes converter with OCR, Q&A, and summaries for mental health documentation.

## Technical Summary
- **OCR**: Tesseract.js for browser-based text extraction
- **AI**: @xenova/transformers distilled BERT for on-device Q&A
- **Storage**: Local browser processing, zero data upload
- **Features**: Extractive summaries, keyword generation, translation
- **Performance**: Single-file PWA, works offline after first load

## Impact
Removes friction from mental health documentation, allowing families to digitize notes instantly without privacy concerns. Built for overwhelmed caregivers and patients.

## Quick Start
Visit [live demo](https://mimindmendinc.github.io/snapnote) or:
```bash
git clone https://github.com/MiMindMendinc/SnapNote.git
cd SnapNote
# Open index.html in browser
```
